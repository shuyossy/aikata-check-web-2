# 用語集
- ドキュメント
  - 複数ファイルで構成される
    - 画像も含むことができる
  - これらファイルから抽出したテキスト（画像ファイルの場合はそのまま画像として取り扱う）を合わせたものがドキュメントの内容となる
- チェックリスト
  - 複数のチェック項目で構成される
- チェック項目
  - 単一のチェック観点
- レビュー結果
  - AIによるレビューの結果
    - 一つのチェック項目に対して一つのレビュー結果が紐づく
  - 以下で構成される
    - コメント
      - ユーザが指定したフォーマットでAIがコメントを生成
    - 評定
      - ユーザが指定したラベル（A,B,C,...等）とラベル定義に従ってAIが評定をつける
- チェック対象
  - AIレビューの対象
  - 複数ファイルで構成される
    - ドキュメントやその他形式（プラグインでAIKATAが処理可能なように変換する想定）として取り扱う
- レビュースペース
  - チャットアプリに当てはめると、チャットルームに相当する概念
  - 単一のチェックリストに対して複数のレビュー対象を管理する
- プロジェクト
  - 複数のレビュースペースで構成される
  - 複数ユーザを登録することができ、プロジェクトに登録されているユーザはプロジェクト内のレビュースペースにアクセスしてレビューの実行やレビュー結果の参照が可能となる
  - ユーザは複数のプロジェクトに所属することが可能
- レビュー設定
  - AIによるレビュー実行時にユーザがアレンジ可能な設定項目
  - 一つのレビュー対象に対して一つのレビュー設定が紐づく
  - 以下で構成される
    - 追加指示（AIレビュー実行時のシステムプロンプトに組み込まれる）
    - 同時レビュー項目数
      - AIのレビュー実行時に一度にいくつのチェック項目をレビューさせるか指定
    - コメントフォーマット
      - レビュー結果に含まれるコメントのフォーマット
    - 評定基準
      - 以下ペアのリストで構成される
        - ラベル（例：A）
        - ラベル定義（例：チェック項目の要件を満たしており、問題がない状態）
- チェックリスト登録
  - ファイル抽出やAIチェックリスト生成や手動でチェックリストをレビュースペースに登録すること
  - ファイル抽出
    - チェックリストファイルからチェックリスト項目をそれぞれ機械的に抽出する
  - AIチェックリスト生成
    - ドキュメントとユーザが指定したチェックリスト作成要件をAIに与えて、チェックリストを生成させる
- 少量レビュー
  - レビュー対象（厳密にはレビュー対象と、レビューを実行するためのチェック項目やシステムプロンプトも含む）がAIのコンテキストウィンドウに収まる場合のレビュー方法
  - AIのコンテキストにレビュー対象をそのまま与えてレビューさせる
- 大量レビュー
  - レビュー対象がAIのコンテキストウィンドウに収まらない場合のレビュー方法
  - レビュー対象を分割し、それぞれをAIにレビューさせた上で、最終的にこれらのレビュー結果をレビュー結果統合用のAIでマージする
- レビュー対象キャッシュ
  - レビュー実行のためにアップロード
