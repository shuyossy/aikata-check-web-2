レビュー設定

チェック項目に加えて、AIが考慮すべき追加の観点を指定できます

項目

AIが同時に処理するチェック項目の数を指定します (推奨: 3-5項目)

3

評定基準

AIが各チェック項目を評価する際の基準を確認してください

A

適合

チェック項目の要件を満たしており、問題がない状態

B

要改善

軽微な問題や改善の余地があるが、致命的ではない状態

C

不適合

重大な問題があり、必ず修正が必要な状態

-

評価不可

ドキュメント内に該当する情報が見つからず、評価できない状態

キャンセル

レビュー実行のヒント

  • ファイルアップロード後、すぐにレビューが開始されます
  • レビュー中も途中経過を確認できます
  • 追加指示でAIに特定の観点を重視させることができます
  • 同時レビュー項目数を増やすと処理が速くなりますが、精度が若干低下する可能性があります
