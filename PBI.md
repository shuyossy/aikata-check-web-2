# ID: 1
- PBI名: レビュー対象削除時の挙動改善
- ステータス: done
- 受け入れ基準
  - レビュー対象一覧画面から削除する際、以下のような挙動になっている
    - レビュー実行中の場合、workflow実行がキャンセルされて、終了
      - キャンセル処理中はキューイングされている次のタスクは実行されない
    - レビュー実行前（キューイング中）はキューから削除される
  - レビュー対象を削除する際は、そこに紐づけられているドキュメントキャッシュ、大量レビュー結果キャッシュも削除される
    - 横展開して、プロジェクトやレビュースペースを削除した際も、紐づくレビュー対象がカスケード削除され、各種キャッシュも削除されることを確認すること
- 注意事項
  - workflowのキャンセル方法はmastraのドキュメントを参照すること
