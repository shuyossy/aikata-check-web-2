# ID: 1
- PBI名: サイドバーからもレビューが実行できるようにする
- ステータス: done
- ユーザストーリー
  - ユーザはサイドバーからも新規レビューを実行できる
  - 現状は、レビュースペースのホーム画面でしか新規レビューを実行できないので、レビューを実行したい際に、直ぐに実行できず（どこからレビュー実行できるのか迷う）、UXを損ねているのだ
- 受け入れ基準
  - サイドバーから新規レビューが実行できる
- 実装内容
  - サイドバーのレビュースペース展開セクション内に「新規レビュー」リンクを追加
  - チェックリストリンクの直下に配置し、緑色でアクション可能であることを視覚的に強調
  - リンク先は既存の `/projects/{projectId}/spaces/{spaceId}/review/new` ページ
