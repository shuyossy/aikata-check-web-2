# ID: 1
- PBI名: チェックリスト項目編集の挙動改善
- ステータス: done
- 背景
  - チェックリスト項目の削除が即時反映されてしまう
- 受け入れ基準
  - チェックリスト項目の編集（追加、編集、削除）した内容については、「変更を保存」ボタンが押下されるまで保存されないこと

# ID: 1
- PBI名: レビュー時の各種キャッシュの取り扱いに問題がないか点検する
- ステータス: to do
- 背景
  - 本システムはキャッシュの取り扱いが複雑なので問題ないか確認する必要がある
- 受け入れ基準
  - 以下のようにキャッシュが取り扱えているか確認し、問題あれば、修正されていること
    - ドキュメントキャッシュ
      - 初回レビュー実行時に保存され、レビュー対象削除と共にキャッシュも削除される
        - レビュー対象が削除され得るシーン
          - レビュー対象一覧から直接削除
          - プロジェクトやレビュースペース単位での削除
      - リトライ時には保存済みのドキュメントキャッシュが参照されている
      - QA時のドキュメント調査時にドキュメントキャッシュが参照されている
    - 大量レビュー結果キャッシュ
      - 大量レビュー実行時に、個別レビューの結果がキャッシュされている
      - リトライ実行時にリトライ対象の大量レビュー結果キャッシュが削除されている（特に前回が大量レビューだった場合は必須）
        - 失敗項目のみリトライの場合は、失敗項目の大量レビューキャッシュが削除されている
        - 前項目リトライの場合は全ての項目の大量レビューキャッシュが削除されている
      - QA時にQA対象チェックリスト項目の大量レビューキャッシュが参照されている
